# Gerber Files

Place your PCB fabrication output here, e.g.:

- `gerber.zip` — the full Gerber/drill file package exported from
  EasyEDA, ready to upload directly to a fab house (JLCPCB, PCBWay,
  etc.)

If you'd rather version-control the individual files instead of a
zip, extract `gerber.zip` into this folder so each `.gbr`/`.drl`
file is tracked individually.
