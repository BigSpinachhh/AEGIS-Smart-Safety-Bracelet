# Schematic

Place your EasyEDA schematic exports here, e.g.:

- `schematic.png` — full schematic image (referenced by the main README)
- `schematic.pdf` — printable version (optional)
- Any EasyEDA project export files (`.json`, `.eprj`, etc.)

The main `README.md` links to `schematic.png` in this folder —
rename your exported image to match, or update the link.
