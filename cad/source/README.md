# CAD Source Files

Place your editable enclosure CAD files here, e.g.:

- Native CAD project files (Fusion 360 `.f3d`, SolidWorks
  `.sldprt`/`.sldasm`, FreeCAD `.FCStd`, etc.)
- `.step` / `.stp` exports (universal format, recommended for
  sharing across CAD tools)
- Any reference drawings used while modeling the enclosure

See [`docs/ENCLOSURE.md`](../../docs/ENCLOSURE.md) for the
dimensional spec these files implement.
